Story Analysis 2021


Experiment Session

Story Analysis is a dedicated website for performing Cognitive Reading and Writing Experiments. Please add the following participant details for starting a session. Data collected during the experiment will be used for academic research. Please email to storyanalysis@gmail.com for unique user code for participation.

Instructions:

  • 1. Enter the participant details in the following form.
  • 2. After entering the participant details, each participant will be given 4 set of tasks. There are 6 tasks in total.
  • 3. All tasks needs to be completed.
  • 4. Please start one task at a time following the given order.
  • 5. New task can only be started after submission of previous tasks.
  • 6. Please don't use external resources or website for the tasks.
