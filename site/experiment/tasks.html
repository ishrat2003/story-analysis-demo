Tasks

These tasks are designed to understand reading and writing activity by human intelligence based on provided information. Please do the tasks in the given order. The order of the tasks sets are: A, B, C and D. In each set follow the order of the given individual tasks. You should complete all the 6 tasks given in these 4 task sets. For each tasks in set A and B, we advice you to spend approximately 10 mins. For each task in set C and D, we advice to spend approximately 20 mins. The task links will be open in a different tab. After submitting the task, close the tab and come back to this tasks page to pick up the next one in the given order. It is important to follow the exact order of the tasks for this reading and writing experiment.
