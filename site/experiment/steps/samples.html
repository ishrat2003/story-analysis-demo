Story Analysis
Experiment



Tasks

The tasks of this experiment are designed to understand reading and writing activity by human intelligence based on the provided information. You have to do tasks in the given order. There will be 6 tasks in 4 sets (A, B, C and D). The order of the task sets is A, B, C and D. In each set, you have to follow the order of the given individual tasks. You should complete all 6 tasks during the session. Tasks in sets A and B will either have visualization or text followed by a set of questions. You need to provide the answers based on the given information on the page. The task in sets C and D will have a list of documents. One of the sets will have additional visualization. You need to read the provided information to write a story about the given topic. For each task in sets A and B, we advise you to spend approximately 10 mins. For each task in sets C and D, we advise spending approximately 20 mins.

Sample tasks

  • Set A/B (approximately 10 mins/task):
    • Single document task with visualization for understanding story reading
    • Single document task with text for understanding story reading
  • Set C/D (approximately 20 mins/task):
    • Listed documents task with visualization for understanding story writing
    • Listed documents task without visualization for understanding story writing
